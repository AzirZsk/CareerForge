# 项目介绍

## CareerForge 是什么

CareerForge 是一款 **AI 驱动的全流程智能求职助手**，帮助你从简历优化到模拟面试，一站式搞定求职准备。

![CareerForge 界面展示](/images/首页展示.png)

## 核心特性

- **智能简历管理** — 上传 PDF 一键解析为结构化数据，区块式精细编辑，支持多版本管理与简历派生
- **简历优化工作流** — 一键触发三阶段全自动优化：快速诊断 → 生成建议 → 内容改写，实时查看进度，对比确认后一键应用
- **AI 对话式优化** — 悬浮球随时唤起，用自然语言描述优化方向，AI 生成字段级 Diff 修改建议，逐条确认或批量应用
- **简历定制** — 输入目标岗位 JD，AI 自动分析匹配度并生成定制化简历，突出岗位所需的核心技能和经验
- **面试中心** — 从准备到复盘一站式管理，AI 自动调研公司背景、分析 JD 生成准备清单，面试后深度复盘分析
- **AI 语音模拟面试** — 与 AI 面试官实时语音对话，根据简历智能出题与追问，卡壳时可求助，全程录音支持回放复盘

## 适用人群

- 正在求职、需要优化简历的求职者
- 想通过模拟面试提升面试表现的同学
- 希望用 AI 工具提升求职效率的任何人
